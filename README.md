# 猫头鹰书店项目

> 搭建一套springboot与springcloud的项目。

## 开发目的

> 公司要从dubbo框架切换到springcloud框架，开发该项目用于技术调研。

> 其中只实现了一些最基本的业务。

> 重在将一些技术以Demo的形式组合起来。

## 该项目使用了哪些技术

### 后端：

> 基于springboot与springcloud开发分布式服务；

> 在springboot框架中使用orm持久化、redis缓存、rabbitmq消息队列、nosql数据库；

### 前端：

> 基于webpack + vue2 + vuex2 + axios + vue-router2

> pc端使用elementui，移动端自己编写bootstrap响应式框架
